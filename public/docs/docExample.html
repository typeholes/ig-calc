Type sin(x) in the highlight box (in green) The graph for sin(x) should appear above New options will appear below you can use the color picker to change the color of the graph. Click the add button and sin(x) expression will move to the list of expressions at the top left If there are any free variables in an expression they will be listed below the expression
Click the dx button next to the free variable x to set the input expression to the derivative by x
You can then add the derivative to the expression list by clicking the add button
